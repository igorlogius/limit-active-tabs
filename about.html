Limits the number of active tabs, by unloading older tabs (determined by last tab activation) Short Demo Video: https://github.com/igorlogius/limit-active-tabs/assets/67047467/9dc448e6-f598-42cd-ba15-637bbfdbfa90 Why should i use this and not "Auto Tab Discard"? This addon is more lightweight and far easier to use (configure), since has only a few options. Usage/Features:
  1. change the max number of active tabs (default is 3) in the preferences
  2. change if pinned tabs should be included when considering unloading tabs
  3. use the toolbar button to exclude the current or all highlighted tabs from unloading
